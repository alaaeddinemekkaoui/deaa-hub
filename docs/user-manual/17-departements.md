# Departements

![Departements](../screenshots/17-departments.png)

**Lien:** `/departments`

## Objectif

La page Departements gere les structures internes de l'etablissement.

## Utilisation

- Creer un departement.
- Modifier le nom ou le code.
- Consulter les filieres rattachees.
- Supprimer un departement inutilise.

## Points importants

- Les departements servent au filtrage des donnees et aux droits d'acces.
- Un departement utilise par des filieres ou utilisateurs ne doit pas etre supprime sans verification.
- Les utilisateurs peuvent etre rattaches a un ou plusieurs departements.
