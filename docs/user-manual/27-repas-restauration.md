# Repas restauration

![Repas restauration](../screenshots/27-settings-restauration.png)

**Lien:** `/settings/restauration`

## Objectif

Cette page configure les repas, prix et disponibilites de la restauration.

## Utilisation

- Ajouter un repas.
- Definir le prix.
- Activer ou desactiver un repas.
- Modifier les informations d'un repas existant.

## Points importants

- Les repas actifs sont visibles dans le module Restauration.
- Les prix doivent etre verifies avant l'ouverture des reservations.
- Les modifications peuvent impacter les transactions futures.
