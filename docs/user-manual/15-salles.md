# Gestion des salles

![Salles](../screenshots/15-rooms.png)

**Lien:** `/rooms`

## Objectif

La page Salles gere les espaces physiques de l'etablissement.

## Utilisation

- Ajouter une salle.
- Definir la capacite.
- Renseigner les equipements.
- Rattacher la salle a un departement si necessaire.
- Modifier ou supprimer une salle.

## Points importants

- Les salles sont utilisees par l'emploi du temps et les reservations.
- La capacite aide a choisir une salle adaptee.
- Le rattachement departement peut limiter les droits d'approbation.
