# Ressources cours

![Ressources cours](../screenshots/10-cours-resources.png)

**Lien:** `/cours-resources`

## Objectif

La page Ressources cours permet de partager et gerer les supports pedagogiques.

## Utilisation

- Filtrer par classe, module ou cours.
- Ajouter un fichier de cours.
- Modifier les informations de la ressource.
- Telecharger ou ouvrir une ressource.
- Supprimer les ressources obsoletes selon les droits.

## Points importants

- Les fichiers doivent etre nommes clairement.
- Les ressources sont utiles pour centraliser polycopies, PDF, images et supports.
- Les enseignants et etudiants peuvent avoir une visibilite differente selon le role.
