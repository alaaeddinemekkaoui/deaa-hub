# Etudiants

![Etudiants](../screenshots/03-students.png)

**Lien:** `/students`

## Objectif

La page Etudiants centralise les dossiers des apprenants.

## Utilisation

- Rechercher un etudiant par nom, code Massar, CIN ou email.
- Filtrer par departement, filiere, cycle, option ou classe.
- Ajouter un etudiant manuellement.
- Importer une liste d'etudiants depuis un fichier.
- Modifier, consulter ou supprimer un dossier selon les droits.
- Ouvrir le profil detaille d'un etudiant.

## Points importants

- La suppression doit rester reservee aux administrateurs.
- Les utilisateurs lies a un departement voient uniquement le perimetre autorise.
- Garder les informations Massar, CIN, classe et filiere a jour facilite les deliberations et les exports.
