# Tableau de bord

![Tableau de bord](../screenshots/02-dashboard.png)

**Lien:** `/dashboard`

## Objectif

Le tableau de bord donne une vue generale de l'activite academique et administrative.

## Utilisation

- Consulter les indicateurs globaux: etudiants, enseignants, classes, filieres, documents et departements.
- Lire les graphiques de repartition pour comprendre la distribution des effectifs.
- Verifier l'activite recente.
- Utiliser les actions rapides pour rejoindre les modules frequents.

## Points importants

- Les donnees affichees peuvent etre limitees par departement selon le role.
- Le bouton **Statistiques** permet d'acceder aux exports.
- C'est la page de controle quotidienne pour l'administration.
