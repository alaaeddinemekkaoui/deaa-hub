# Restauration

![Restauration](../screenshots/22-restauration.png)

**Lien:** `/restauration`

## Objectif

La page Restauration gere les repas, soldes, reservations, tickets et transactions.

## Utilisation

- Consulter les repas disponibles.
- Rechercher un etudiant.
- Crediter ou ajuster un solde selon les droits.
- Reserver un repas.
- Emettre ou consulter un ticket.
- Suivre les transactions.

## Points importants

- Le role `restauration` peut etre utilise pour limiter l'acces aux operations de cantine.
- Les tickets et transactions doivent rester traces.
- Les prix et repas actifs se configurent dans les parametres restauration.
