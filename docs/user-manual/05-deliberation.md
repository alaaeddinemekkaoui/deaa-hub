# Deliberation

![Deliberation](../screenshots/05-deliberation.png)

**Lien:** `/deliberation`

## Objectif

La page Deliberation permet de consolider les resultats d'une classe et de preparer les decisions academiques.

## Utilisation

- Choisir la classe ou le contexte academique.
- Consulter les moyennes, resultats et statuts.
- Verifier les details avant validation.
- Exporter ou produire les releves lorsque l'action est disponible.

## Points importants

- La qualite des deliberations depend des notes saisies dans **Epreuves**.
- Toujours verifier les filtres avant de traiter une classe.
- Les decisions doivent etre validees par les responsables habilites.
