# Grades enseignants

![Grades enseignants](../screenshots/30-settings-teacher-grades.png)

**Lien:** `/settings/teacher-grades`

## Objectif

Cette page configure les grades academiques des enseignants.

## Utilisation

- Ajouter un grade.
- Modifier le libelle.
- Supprimer un grade inutilise.
- Affecter le grade depuis la fiche professeur.

## Points importants

- Exemples: professeur habilite, maitre assistant, assistant, vacataire.
- Les grades facilitent les filtres et rapports RH.
- Eviter les variantes orthographiques pour le meme grade.
