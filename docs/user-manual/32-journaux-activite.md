# Journaux d'activite

![Journaux d'activite](../screenshots/32-activity-logs.png)

**Lien:** `/activity-logs`

## Objectif

Les journaux d'activite affichent les actions importantes effectuees dans l'application.

## Utilisation

- Consulter les actions recentes.
- Filtrer par utilisateur, module ou type d'action si disponible.
- Identifier qui a cree, modifie ou supprime une donnee.

## Points importants

- Les journaux servent a l'audit et au suivi interne.
- Ils doivent etre accessibles uniquement aux administrateurs ou responsables habilites.
- Les actions sensibles doivent etre tracees clairement.
