# Demandes administratives

![Demandes](../screenshots/23-workflows.png)

**Lien:** `/workflows`

## Objectif

La page Demandes suit les workflows administratifs et les demandes de documents.

## Utilisation

- Creer une demande.
- Assigner un responsable.
- Suivre le statut.
- Ajouter les informations ou documents necessaires.
- Cloturer la demande quand le traitement est termine.

## Points importants

- Les demandes permettent de tracer les traitements administratifs.
- Les statuts doivent etre tenus a jour.
- Cette page est generalement reservee a l'administration et au staff.
