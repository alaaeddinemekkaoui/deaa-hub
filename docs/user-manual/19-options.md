# Options

![Options](../screenshots/19-structure-options.png)

**Lien:** `/structure`

## Objectif

La page Options gere les specialites ou parcours rattaches aux filieres.

## Utilisation

- Creer une option.
- Lier l'option a une filiere.
- Modifier ou supprimer une option.
- Consulter les classes rattachees.

## Points importants

- Une option appartient a une filiere.
- Les options facilitent les filtres dans les classes, etudiants et notes.
- Supprimer une option utilisee peut rendre des donnees incoherentes.
