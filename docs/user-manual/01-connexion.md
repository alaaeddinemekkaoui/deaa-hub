# Connexion

![Connexion](../screenshots/01-login.png)

**Lien:** `/login`

## Objectif

La page de connexion permet d'acceder a DEAA-Hub avec un identifiant et un mot de passe.

## Utilisation

1. Saisir le username ou l'adresse email.
2. Saisir le mot de passe.
3. Cliquer sur **Se connecter**.

## Points importants

- Les champs sont obligatoires.
- Apres connexion, l'utilisateur est redirige vers le tableau de bord.
- Les menus visibles dependent du role de l'utilisateur.
- En cas d'echec, verifier les identifiants ou contacter un administrateur.
