# Professeurs

![Professeurs](../screenshots/08-teachers.png)

**Lien:** `/teachers`

## Objectif

La page Professeurs gere les enseignants permanents, vacataires et leurs rattachements.

## Utilisation

- Rechercher par nom, CIN, email ou telephone.
- Filtrer par departement, filiere, role ou grade.
- Ajouter un enseignant.
- Modifier les informations administratives.
- Consulter le profil d'un enseignant.
- Importer ou exporter les donnees si les boutons sont disponibles.

## Points importants

- Les roles et grades enseignants sont configures dans les parametres.
- Les rattachements departement/filiere facilitent la planification des cours.
- Les suppressions doivent etre controlees pour ne pas casser l'historique.
