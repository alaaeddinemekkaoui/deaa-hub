# Types de documents

![Types de documents](../screenshots/28-settings-document-types.png)

**Lien:** `/settings/document-types`

## Objectif

Cette page configure les categories de documents administratifs.

## Utilisation

- Creer un type de document.
- Modifier son libelle.
- Supprimer un type inutilise.
- Utiliser ces types dans les demandes ou dossiers.

## Points importants

- Les types de documents rendent les dossiers plus faciles a controler.
- Eviter les libelles proches ou redondants.
- Ne pas supprimer un type deja utilise sans migration des dossiers.
