# Epreuves

![Epreuves](../screenshots/04-grades.png)

**Lien:** `/grades`

## Objectif

La page Epreuves sert a saisir et suivre les notes par classe, module et element.

## Utilisation

- Selectionner les filtres academiques jusqu'a la classe ou l'element voulu.
- Creer ou modifier une epreuve.
- Saisir les notes des etudiants.
- Importer des notes quand l'interface le permet.
- Controler les notes avant deliberation.

## Points importants

- Verifier la classe et le module avant toute saisie.
- Les notes incoherentes doivent etre corrigees avant l'export ou la deliberation.
- Les droits de saisie peuvent dependre du role utilisateur.
