# Cycles

![Cycles](../screenshots/20-cycles.png)

**Lien:** `/cycles`

## Objectif

La page Cycles gere les grands niveaux de formation.

## Utilisation

- Creer un cycle academique.
- Modifier son nom ou code.
- Supprimer un cycle inutilise.

## Points importants

- Les cycles sont utilises pour classer les etudiants et programmes.
- Exemples: preparatoire, ingenieur, veterinaire.
- Garder des libelles courts et constants ameliore les statistiques.
