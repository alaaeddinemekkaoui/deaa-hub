# Filieres

![Filieres](../screenshots/18-filieres.png)

**Lien:** `/filieres`

## Objectif

La page Filieres gere les programmes de formation.

## Utilisation

- Ajouter une filiere.
- Rattacher la filiere a un departement.
- Modifier son nom, code ou description.
- Suivre les classes et options associees.

## Points importants

- Les filieres structurent les etudiants, classes, options et statistiques.
- Eviter les codes dupliques.
- Changer une filiere peut impacter les filtres de nombreuses pages.
