# Transferts

![Transferts](../screenshots/07-transfers.png)

**Lien:** `/transfers`

## Objectif

La page Transferts gere le passage des etudiants d'une classe a une autre ou d'une annee academique a la suivante.

## Utilisation

- Choisir la classe source.
- Selectionner les etudiants concernes.
- Definir le statut: admis, non admis, redoublant ou autre statut prevu.
- Choisir la classe cible et l'annee academique.
- Valider le transfert.

## Points importants

- Verifier les listes avant validation.
- Les transferts impactent l'historique academique.
- Une mauvaise classe cible peut fausser les effectifs et les statistiques.
