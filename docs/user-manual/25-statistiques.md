# Statistiques

![Statistiques](../screenshots/25-statistics.png)

**Lien:** `/statistics`

## Objectif

La page Statistiques sert a consulter les donnees consolidees et exporter des rapports.

## Utilisation

- Choisir une categorie de donnees.
- Appliquer les filtres disponibles.
- Consulter les indicateurs.
- Exporter les donnees en CSV ou autre format propose.

## Points importants

- Les exports sont utiles pour les rapports de direction.
- Verifier les filtres avant export.
- Les donnees peuvent etre filtrees par role et departement.
